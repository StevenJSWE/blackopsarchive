# Research Scratchpad

Working directory for Black Ops Archive research. Nothing here is published — it feeds into articles.

```
research/
  operations/   # Per-operation research notes, source lists, raw facts
  sources/      # Saved source links, FOIA doc indexes, reading lists
  notes/        # General notes, ideas, content strategy
  drafts/       # WIP article drafts before moving to src/content/articles/
```

## Usage

- Drop raw research here before writing an article
- Use `operations/[slug].md` as a scratchpad per operation
- Commit this folder — it's useful context for future sessions
